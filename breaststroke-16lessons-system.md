# 📚 蛙泳完整教学体系（16课时）
> 适合所有年龄段，从零基础到精通入门的完整学习路径

---

## 🎯 课程理念（核心原则）

| 原则 | 说明 |
|------|------|
| **水性第一** | 不着急教动作，先让学员"不怕水、享受水" |
| **感觉驱动** | 强调"找感觉"，而不是机械模仿动作 |
| **分解细化** | 每个动作拆到不能再拆，循序渐进 |
| **纠错前置** | 提前告诉学员"会犯什么错"，而不是犯错后纠正 |

## 💡 教学小贴士

### 1. 如何让学员不紧张？
- 前2节课先"玩水"，打水仗都行
- 永远说"我们来试试这个"，不说"你必须做到"
- 学员做对了立刻说"对！就是这个感觉！"

### 2. 最常见的 3 个问题怎么纠正？

| 问题 | 纠正方法 |
|------|----------|
| **用鼻子吸气呛水** | 强迫学员：抬头就张嘴，低头鼻子就吐泡泡 |
| **蹬腿不走** | 让学员趴在池边，用手感受他的翻脚和蹬夹 |
| **游着游着沉了** | 告诉他"飘的时间再长一点"，不要急着做下一个动作 |

### 3. 每节课必说的三句话
1. "放松，水会托着你"
2. "慢一点，不用急"
3. "找感觉，不是用蛮力"

## 📹 学员成长记录建议

### 视频记录方案
| 记录节点 | 拍摄内容 | 用途 |
|----------|----------|------|
| **每节课** | 30秒短视频：当天学习的重点动作 | 课后发给学员，对比进步 |
| **里程碑**（第4/8/12/16课） | 1分钟完整视频：测试全程 | 结业时做成成长合集 |
| **对比视频** | 第5课 vs 第8课的打腿对比 | 让学员直观看到自己的进步 |

### 飞书文档使用建议
1. **每个学员一个飞书文档**
2. 文档结构：
   - 学员基本信息
   - 学习进度打卡表（上面的表格直接复制）
   - 每节课视频记录（嵌入视频或放视频链接）
   - 教练评语区
3. **分享给学员家长/本人**，让他们随时可以看进度

### 学员成就感提升技巧
1. **每4课颁发一张电子证书**（水性达标、腿部达标、入门达标、结业证书）
2. **对比视频**：第1课 vs 第16课，变化是最好的激励
3. **学员墙**：所有结业学员的照片和50米成绩展示
4. **打卡积分**：每完成一课积1分，积分可以兑换小礼物
